CSS 기본 선택자

선택자란?
특정 HTML 요소를 선택하고자할 때 사용하는 기능
해당 요소를 선택해서 원하는 스타일과 기능을 적용 가능

1. 모든 선택자 (*)

해당 문서상의 모든 요소들을 선택 
        <style>
        * {
            스타일 속성:값;
            스타일 속성:값;
        }
        </style>

2. 태그 선택자 (태그명)

해당 문서상의 같은 태그들을 선택 
        <style>
        tag {
            스타일 속성:값;
            스타일 속성:값;
        }
        </style>

3. 클래스 선택자 (.클래스명)

해당 문서내에 내가 원하는 여러 요소를 선택 
        <style>
        .class {
            스타일 속성:값;
            스타일 속성:값;
        }
        </style>
